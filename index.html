Installation

Firefox

  1. Download the .xpi file from the releases page
  2. Open Firefox and go to about:addons
  3. Click the gear icon and select "Install Add-on From File"
  4. Select the downloaded .xpi file

Chrome

  1. Download the .zip file from the releases page
  2. Extract the zip file to a folder
  3. Open Chrome and go to chrome://extensions
  4. Enable "Developer mode" (toggle in top right)
  5. Click "Load unpacked" and select the extracted folder

How to Use

  1. Select Athletes: Click the + icon next to athlete names in your feed
  2. Create Groups: Open the sidebar (üë• button) and create a new group
  3. Add to Groups: Add selected athletes to your groups
  4. Filter Feed: Use the dropdown at the top of your feed to filter by group
  5. Manage Groups: Edit, delete, or reorganize groups in the sidebar
